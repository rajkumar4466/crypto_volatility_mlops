  <files>docker-compose.yml</files>
  <action>
    Create `docker-compose.yml` at the project root with the following services:

    **postgres:**
    - Image: postgres:16 (matches RDS engine_version in Terraform)
    - Environment: POSTGRES_USER=airflow, POSTGRES_PASSWORD=airflow, POSTGRES_DB=airflow
    - Ports: 5432:5432
    - Volume: postgres_data:/var/lib/postgresql/data
    - Healthcheck: pg_isready -U airflow (interval 10s, timeout 5s, retries 5)

    **redis:**
    - Image: redis:7-alpine (matches ElastiCache engine_version 7.1)
    - Ports: 6379:6379
    - Command: redis-server --appendonly yes
    - Healthcheck: redis-cli ping (interval 10s, timeout 5s, retries 5)

    **airflow-init:**
    - Image: apache/airflow:3.1.8
    - depends_on: postgres (healthy), redis (healthy)
    - Environment: AIRFLOW__DATABASE__SQL_ALCHEMY_CONN=postgresql+psycopg2://airflow:airflow@postgres/airflow
    - Entrypoint: /bin/bash
    - Command: -c "airflow db migrate && airflow users create --username admin --password admin --firstname Admin --lastname User --role Admin --email admin@example.com || true"
    - profiles: ["init"] — only runs when explicitly called with `docker compose --profile init up airflow-init`

    **airflow-scheduler:**
    - Image: apache/airflow:3.1.8
    - depends_on: postgres (healthy), redis (healthy)
    - Environment:
      - AIRFLOW__DATABASE__SQL_ALCHEMY_CONN=postgresql+psycopg2://airflow:airflow@postgres/airflow
      - AIRFLOW__CORE__EXECUTOR=SequentialExecutor (matches t3.micro production constraint)
      - AIRFLOW__CORE__LOAD_EXAMPLES=false
      - AIRFLOW__CORE__DAGS_FOLDER=/opt/airflow/dags
    - Volumes: ./dags:/opt/airflow/dags, ./logs:/opt/airflow/logs, ./src:/opt/airflow/src
    - Command: scheduler
    - Restart: unless-stopped

    **airflow-api-server:**
    - Image: apache/airflow:3.1.8
    - depends_on: postgres (healthy)
    - Environment:
      - AIRFLOW__DATABASE__SQL_ALCHEMY_CONN=postgresql+psycopg2://airflow:airflow@postgres/airflow
      - AIRFLOW__CORE__EXECUTOR=SequentialExecutor
      - AIRFLOW__CORE__LOAD_EXAMPLES=false
    - Ports: 8080:8080
    - Volumes: ./dags:/opt/airflow/dags, ./logs:/opt/airflow/logs
    - Command: api-server
    - Restart: unless-stopped

    **Volumes:**
    - postgres_data (named volume)

    **Important notes in comments at top of file:**
    - First run: `docker compose --profile init up airflow-init` to initialize the database and create admin user
    - Then: `docker compose up -d` to start all services
    - Airflow UI: http://localhost:8080 (admin/admin)
    - Redis (Feast online store): localhost:6379
    - PostgreSQL: localhost:5432 (airflow/airflow)

    Do NOT use `version: '3.8'` key — it is obsolete in modern Docker Compose and generates a warning.
