Create `training/train.py` as a standalone executable training script. It must implement `run_training()` returning `(onnx_path: str, metrics: dict, params: dict)`.

**Step-by-step implementation:**

**1. Imports and converter registration (at module level, before any function):**
```python
from onnxmltools import convert_sklearn
from onnxmltools.utils import save_model as save_onnx_model
from onnxmltools.convert.xgboost.operator_converters.XGBoost import convert_xgboost
from skl2onnx import update_registered_converter
from skl2onnx.common.data_types import FloatTensorType
from xgboost import XGBClassifier

update_registered_converter(
    XGBClassifier,
    "XGBoostXGBClassifier",
    convert_xgboost,
    parser=None,
)
```
This MUST happen at module level — not inside a function. Missing this causes `MissingConverter` at conversion time.

**2. Feature loading from Feast offline store:**
```python
from feast import FeatureStore

FEATURE_NAMES = [
    "btc_features:volatility_10m", "btc_features:volatility_30m",
    "btc_features:volatility_ratio", "btc_features:rsi_14",
    "btc_features:volume_spike", "btc_features:volume_trend",
    "btc_features:price_range_30m", "btc_features:sma_10_vs_sma_30",
    "btc_features:max_drawdown_30m", "btc_features:candle_body_avg",
    "btc_features:hour_of_day", "btc_features:day_of_week",
]
FEATURE_COLS = [f.split(":")[1] for f in FEATURE_NAMES]  # strip view prefix
```

Load features via `store.get_historical_features(entity_df, features=FEATURE_NAMES).to_df()`. The `entity_df` must have `entity_id` and `event_timestamp` columns. Get timestamps from the offline store's Parquet file listing or pass them as a parameter.

**3. Time-ordered train/test split (NO shuffle):**
```python
cutoff = int(len(df) * 0.8)
X_train = df[FEATURE_COLS].iloc[:cutoff].astype(np.float32)
X_test  = df[FEATURE_COLS].iloc[cutoff:].astype(np.float32)
y_train = df["label"].iloc[:cutoff]  # 0=CALM, 1=VOLATILE
y_test  = df["label"].iloc[cutoff:]
```
NEVER use `train_test_split` with `shuffle=True` (default). Time-ordered split only.

**4. GridSearchCV with TimeSeriesSplit:**
```python
from sklearn.model_selection import GridSearchCV, TimeSeriesSplit

param_grid = {
    "max_depth": [3, 5, 7],
    "learning_rate": [0.01, 0.1, 0.3],
    "n_estimators": [50, 100, 200],
    "subsample": [0.8, 1.0],
}
tscv = TimeSeriesSplit(n_splits=5)
clf = XGBClassifier(
    use_label_encoder=False,
    eval_metric="logloss",
    n_jobs=1,        # CRITICAL: avoid OOM on t3.micro
    random_state=42,
)
grid_search = GridSearchCV(clf, param_grid, cv=tscv, scoring="f1", n_jobs=1, verbose=1)
grid_search.fit(X_train, y_train)
```

**5. Metrics computation:**
```python
from sklearn.metrics import accuracy_score, f1_score, roc_auc_score

y_pred = grid_search.predict(X_test)
y_prob = grid_search.predict_proba(X_test)[:, 1]
metrics = {
    "accuracy": float(accuracy_score(y_test, y_pred)),
    "f1": float(f1_score(y_test, y_pred)),
    "roc_auc": float(roc_auc_score(y_test, y_prob)),
}
```

**6. ONNX export — export best_estimator_, NOT grid_search:**
```python
n_features = X_train.shape[1]
assert n_features == 12, f"Expected 12 features, got {n_features}"
initial_type = [("X", FloatTensorType([None, n_features]))]

onnx_model = convert_sklearn(
    grid_search.best_estimator_,   # NOT grid_search itself
    initial_types=initial_type,
    target_opset=15,
)
onnx_path = "/tmp/model.onnx"
save_onnx_model(onnx_model, onnx_path)
```

**7. Smoke test — call before returning:**
```python
from training.smoke_test import smoke_test_onnx

smoke_test_onnx(onnx_path, n_features=12)  # raises AssertionError on failure
```

**8. Return value:**
```python
return onnx_path, metrics, grid_search.best_params_
```
