Create `training/registry.py` implementing the S3 model registry and promotion gate logic.

**Public interface to implement:**

```python
def promote_or_archive(
    bucket: str,
    run_id: str,
    challenger_f1: float,
    onnx_path: str,
    challenger_metrics: dict,
) -> tuple[str, float]:
    """
    Compare challenger against champion. Promote or archive.
    Returns: (decision: "promoted"|"rejected", champion_f1: float)
    """
```

**Implementation logic:**

1. **Load champion metrics** (handle first-run case):
```python
s3 = boto3.client("s3")
try:
    obj = s3.get_object(Bucket=bucket, Key="models/current_metrics.json")
    champion = json.loads(obj["Body"].read())
    champion_f1 = float(champion["f1"])
    champion_run_id = champion.get("run_id", "unknown")
except s3.exceptions.ClientError as e:
    if e.response["Error"]["Code"] == "NoSuchKey":
        champion_f1 = 0.0          # first run: always promote
        champion_run_id = None
    else:
        raise
```

2. **Promotion branch:**
```python
if challenger_f1 > champion_f1:
    # Archive current model before overwriting (skip gracefully if no current)
    if champion_run_id:
        try:
            s3.copy_object(
                Bucket=bucket,
                CopySource={"Bucket": bucket, "Key": "models/current.onnx"},
                Key=f"models/v{champion_run_id}.onnx",
            )
        except Exception:
            pass  # no current.onnx on first run

    # Write new current model
    s3.upload_file(onnx_path, bucket, "models/current.onnx")

    # Write new champion metrics
    s3.put_object(
        Bucket=bucket,
        Key="models/current_metrics.json",
        Body=json.dumps({
            "f1": challenger_f1,
            "run_id": run_id,
            "metrics": challenger_metrics,
            "timestamp": datetime.utcnow().isoformat(),
        }),
        ContentType="application/json",
    )
    decision = "promoted"
```

3. **Rejection branch:**
```python
else:
    # Archive challenger without replacing current
    s3.upload_file(onnx_path, bucket, f"models/v{run_id}.onnx")
    decision = "rejected"
```

4. **Log promotion decision to S3** (both branches):
```python
promotion_record = {
    "decision": decision,
    "run_id": run_id,
    "challenger_f1": challenger_f1,
    "champion_f1": champion_f1,
    "timestamp": datetime.utcnow().isoformat(),
    "challenger_metrics": challenger_metrics,
}
s3.put_object(
    Bucket=bucket,
    Key=f"runs/{run_id}/promotion.json",
    Body=json.dumps(promotion_record),
    ContentType="application/json",
)
return decision, champion_f1
```

1. wandb.init()
2. Feast feature pull
3. train/test split
4. GridSearchCV fit
5. metrics computation

8. wandb.log(metrics + best_params)

14. wandb.finish() (in finally)
